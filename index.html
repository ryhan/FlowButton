Build Applications with Flow

What's Flow?

Flow is a cross platform task management application created by Metalab. It's a simple way to make sure that everyone on your team knows what's going on, and what they need to do.

Frictionless Task Creation

The process of adding tasks should be frictionless - otherwise, people will tend to keep tasks in their head, or worse, forget about something entirely.

Instead of requiring readers to open a separate application to add a task, you can relatively easily prompt readers to save a task for later using this simple Flow button generator.

It's easy to implement, and makes sure that readers remember to get things done.
